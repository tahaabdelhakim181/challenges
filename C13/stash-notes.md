# Git Stash Notes

## What I did
Modified a file mid-way, needed to switch branches urgently.
Used git stash to shelve the work, switched branches, came back and applied.

## Commands used
  git stash          → saves current changes
  git stash list     → shows saved stashes
  git stash pop      → re-applies latest stash and removes it
  git stash apply    → re-applies without removing from stash list
